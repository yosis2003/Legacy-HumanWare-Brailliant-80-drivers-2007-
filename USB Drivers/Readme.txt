USB driver for BAUM Braille devices
===================================

This file contains Microsoft certified USB drivers for BAUM Braille devices 
SuperVario, VarioConnect and VarioPro (incl. modules) series. Vario 40 USB adapter is also included. 

The USB drivers provide a virtual COM port as well as direct access drivers and run on Microsoft Windows XP, VISTA and Windows 7 32 bit or 64 bit Operating systems. 

To install the drivers run setup.exe !before! the Braille device is connected to the PC. Once installed, the Braille device will automatically be detected and installed when plugged in, no matter on which USB port. Please make sure the Braille display is setup for USB communication port.

This driver replaces older version of BAUM USB drivers.
